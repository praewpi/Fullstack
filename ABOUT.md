# About This Project

## Overview

This repository is a collection of exercises and projects completed as part of the **Full Stack Open** course, a free online course provided by the **Department of Computer Science at the University of Helsinki**.

🔗 [https://fullstackopen.com/](https://fullstackopen.com/)

---

## What Is Full Stack Open?

Full Stack Open is a comprehensive, hands-on course that teaches modern full stack web development. The course focuses on building real-world web applications using the JavaScript ecosystem — from the browser all the way to the server and database.

> *"Learn React, Redux, Node.js, MongoDB, GraphQL and TypeScript in one go!"*

The course introduces developers to the complete development workflow of **Single Page Applications (SPAs)** and covers both frontend and backend concerns.

---

## What I Learned (Parts 0–6)

### 🌐 Part 0 – Fundamentals of Web Apps
An introduction to the basics of web development, covering:
- How browsers load web pages (HTTP requests, rendering)
- The Document Object Model (DOM)
- AJAX and asynchronous JavaScript
- How modern SPAs differ from traditional web apps

### ⚛️ Part 1 – Introduction to React
Getting started with React:
- Creating components and passing props
- Managing component state with `useState`
- Handling events and re-rendering
- JavaScript fundamentals relevant to React

### 🔌 Part 2 – Communicating with Server
Going deeper with React and connecting to backends:
- Rendering collections and working with lists and keys
- Controlled form inputs
- Fetching data from REST APIs using `axios`
- Adding and deleting data via HTTP methods (GET, POST, PUT, DELETE)
- Styling React apps

### 🖥️ Part 4 – Testing Express Servers, User Administration
Backend development with Node.js:
- Structuring an Express application
- Working with MongoDB via Mongoose
- Unit and integration testing with **Jest** and **Supertest**
- User management: registration and login
- Token-based authentication with **JWT**

### 🧪 Part 5 – Testing React Apps
Bringing frontend testing into the picture:
- Storing tokens in the browser (`localStorage`)
- Displaying login forms and managing authentication state
- Frontend unit testing with **React Testing Library**
- End-to-end testing concepts

### 🗂️ Part 6 – State Management with Redux
Advanced state management:
- Flux architecture and the Redux pattern
- Reducers, actions, and the Redux store
- Redux Toolkit for simplified Redux development
- Async actions and side effects
- Connecting Redux to React components with `react-redux`

---

## Technologies Used

- **React** – Component-based UI library
- **Redux / Redux Toolkit** – Predictable state management
- **Node.js** – JavaScript runtime for the server
- **Express** – Web framework for Node.js
- **MongoDB** – NoSQL document database
- **Mongoose** – MongoDB object modeling for Node.js
- **Axios** – Promise-based HTTP client
- **Jest** – JavaScript testing framework
- **Supertest** – HTTP assertions for backend testing
- **JSON Server** – Mock REST API for development

---

## Purpose

This repository serves as a personal learning record and portfolio piece demonstrating proficiency in modern JavaScript full stack development. Each part of the course builds on the previous, culminating in the ability to design, build, and test complete web applications from scratch.
